# Phase 3：图片解析与识别

本阶段目标：实现 UI 设计图的智能解析，提取控件结构与样式信息。

---

## 任务清单

- [ ] 集成图像处理库 / AI 视觉模型
- [ ] 实现 UI 控件类型识别（按钮、文本、图片、输入框等）
- [ ] 实现控件位置与尺寸检测
- [ ] 提取颜色、字体、间距等样式信息

---

## 详细说明

### 集成图像处理库 / AI 视觉模型

- 选择 OCR 引擎（如 PaddleOCR、Tesseract）
- 选择目标检测模型（如 YOLO、DETR）或调用多模态 LLM API
- 设计图片预处理流水线

### 实现 UI 控件类型识别

- 定义支持的控件类型映射表
- 训练 / 微调检测模型（如需要）
- 实现后处理与置信度过滤

### 实现控件位置与尺寸检测

- 输出控件边界框坐标（x, y, width, height）
- 建立控件间的空间关系（父子、相邻）

### 提取颜色、字体、间距等样式信息

- 提取主色、背景色、文字色
- 识别字体家族与字号
- 计算 padding / margin
